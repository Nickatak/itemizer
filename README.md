# Flask App

A basic Flask application with a template, a generic Item model using SQLAlchemy, an item dashboard, and a database interface for CRUD operations.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python -m src.app`
3. Open http://127.0.0.1:5000/ in your browser.
4. Visit /dashboard to view the item dashboard.

## Troubleshooting

- If port 5000 is in use, Flask will use another port. Check the terminal output.
- Ensure you are in the virtual environment.
